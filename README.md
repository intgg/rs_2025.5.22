# 同声传译项目（v1.0.0）

> Latest release: **v1.0.0**

- `FunASR.py`：实时语音识别
- `translation_module.py`：语言翻译模块
- `edge_TTS.py`：文本转语音
- `simultaneous_translator_app.py`：整合 PyQt5 UI 界面
